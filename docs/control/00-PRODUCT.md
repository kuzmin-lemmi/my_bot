# PRODUCT — MVP "Список важных дел с дожимом"

## Рабочее название
FocusDay / Цели дня / Антизабыватель

## Формат MVP
Single-user (только для одного пользователя — создателя продукта).

## Проблема
Обычные todo-приложения легко игнорируются:
- задачи забываются
- напоминания слишком “тихие”
- нет ощущения дожима до действия

## Основная ценность продукта
Приложение помогает:
1. быстро вносить важные цели (через Telegram и телефон)
2. не забывать их в течение дня
3. настойчиво напоминать до действия пользователя
4. формировать дисциплину (стрик/ритм)
5. вести журнал выполненных целей

## Что важно для пользователя (владелец MVP)
Ценность MVP = всё вместе:
- быстро не забывать задачи
- дожим до выполнения
- дисциплина/стрик
- журнал выполненного

## Core Loop (ядро продукта)
1. Пользователь добавляет цель (Telegram или мобильное приложение).
2. Цель назначается на дату (по умолчанию — сегодня).
3. В приложении на телефоне цель видна в списке дня.
4. Приложение напоминает звуком/уведомлениями по интервалу.
5. Напоминания продолжаются, пока пользователь не сделает действие:
   - Выполнено
   - Отложить
   - Перенести на завтра
   - Отменить
6. Действие фиксируется в журнале.

## Критерий успеха MVP (7–14 дней)
Пользователь каждый день заносит 3–5 целей и закрывает не менее 70%.

## Принципы MVP
- сначала рабочий цикл, потом красота
- минимум функций, максимум пользы
- всё, что не усиливает core loop, откладывается
