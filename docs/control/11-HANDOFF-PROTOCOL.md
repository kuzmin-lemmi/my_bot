# HANDOFF PROTOCOL — строгий протокол передачи между UI (Gemini) и Logic (Claude)

## Роли
- UI Model: Gemini (Flutter UI/UX/client behavior)
- Logic Model: Claude (backend/domain/API/DB/state machine)

## Главные правила
1. Нельзя менять контракт/состояния молча
2. Каждый шаг заканчивается HANDOFF
3. Размер задачи — маленький атомарный шаг
4. Конфликты решаются по приоритету control-файлов

## Формат HANDOFF (обязательный)
## HANDOFF
- Роль: UI | LOGIC
- Шаг / задача:
- Что сделано:
- Что изменено в коде (файлы/модули):
- Какие control-файлы обновлены:
- Что нужно следующей модели:
- Открытые вопросы:
- Риски / ограничения:
- Как проверить вручную (шаги):
- Что НЕ сделано (если осталось):
