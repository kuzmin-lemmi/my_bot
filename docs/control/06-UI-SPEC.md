# UI SPEC — Flutter Android MVP (UI для Gemini)

## Платформа
- Flutter
- Android-first
- MVP без iOS

## Основные экраны (v1)

### 1. Сегодня (главный)
- Заголовок "Сегодня"
- Текущая дата
- Сводка (всего/активных/выполнено/стрик)
- Список карточек целей
- Кнопка "Пауза напоминаний"
- Переход к календарю
- Кнопка добавления цели

#### Карточка цели
- title
- time (если есть)
- note (если есть)
- статус
- "Отложено до HH:MM" для snoozed
- priority badge (минимально)

#### Кнопки (обязательные)
- ✅ Выполнено (с подтверждением)
- ⏰ Отложить (с выбором вариантов)
- ↪ Завтра
- ❌ Отменить

### 2. Календарь (месяц)
- Переключение месяцев
- Сетка месяца
- Индикатор целей по дням
- Выбор даты тапом
- Список целей выбранного дня
- Кнопка "Добавить"

### 3. День (выбранная дата)
- Заголовок с датой
- Список целей
- Добавить цель
- Те же действия по карточкам

### 4. Добавить / Редактировать цель
Поля:
- title (обяз.)
- note
- date
- time
- priority
- save

### 5. Настройки напоминаний
- interval_minutes
- active window (start/end)
- quiet period (toggle + start/end)
- sound on/off
- persistence mode (soft/normal/hard)
- escalation on/off + step
- snooze options
- global pause
- save

### 6. Журнал (минимальный)
- список событий за день
- фильтр по дате (минимум)

## UX-важное
- Главный экран должен быть понятен за 1–2 секунды
- Кнопки действий крупные и очевидные
- Подтверждение для "Выполнено" и "Отменить"
- Без перегруза дизайна
