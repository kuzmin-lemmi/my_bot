# TELEGRAM INPUT RULES — правила ввода целей через Telegram (MVP v1)

## Цель
Максимально быстрый ввод без перегруза командами.

## Минимальная логика v1
Если пользователь отправляет обычный текст без команды:
- создаётся цель на сегодня (target_date = today)

Если сообщение содержит несколько строк:
- каждая непустая строка = отдельная цель (по умолчанию на сегодня)

## Примеры
### 1 цель
Проверить тетради 10А

### Несколько целей
Проверить 8Б
Сделать план урока
Подготовить презентацию

## Команды (минимум v1)
- /start
- /help

## Команды (v1.1 / если быстро получится)
- /today <текст>
- /tomorrow <текст>
- /add YYYY-MM-DD <текст>
- /list
- /done

## Дата по умолчанию
Если дата не указана явно:
- target_date = today

## Парсинг времени/приоритета
В MVP v1 НЕ делать сложный NLP-парсинг.
Telegram = быстрый ввод title.
Время/заметка/приоритет — редактирование в телефоне.

## Валидация
- пустые строки игнорировать
- trim пробелы
- очень длинные строки пока допускаются

## Ответ бота
- короткое подтверждение
- сколько целей добавлено
- на какую дату
- без длинных сообщений
