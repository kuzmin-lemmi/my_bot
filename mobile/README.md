# FocusDay Mobile (Flutter)

MVP приложение для управления целями с настойчивыми напоминаниями.

## Технологии
- **Flutter** (Android-first)
- **Provider** (State Management)
- **http** (API Client)
- **Table Calendar** (Календарь)

## Как запустить
1. Установите Flutter: [https://docs.flutter.dev/get-started/install](https://docs.flutter.dev/get-started/install)
2. Перейдите в папку проекта: `cd mobile`
3. Установите зависимости: `flutter pub get`
4. Настройте подключение к Backend в `lib/main.dart`:
   - Измените `baseUrl` на адрес вашего сервера (например, `http://10.0.2.2:8000` для эмулятора Android)
   - Измените `token` на ваш `MVP_TOKEN` из `.env` бэкенда.
5. Запустите: `flutter run`

## Особенности
- **Главный экран**: список целей на сегодня с кнопками быстрых действий.
- **Действия**:
  - ✅ Выполнено (с подтверждением)
  - ⏰ Отложить (выбор из 10/30/60 мин)
  - ↪ На завтра (перенос даты)
  - ❌ Отменить (с подтверждением)
- **Календарь**: просмотр активности по дням месяца.
- **Настройки**: полное управление политикой напоминаний.
- **Журнал**: история событий (создание, переносы, выполнение).
